# North Texas Housing Market Analysis

This project explores housing market trends in the North Texas region using publicly available data sources.  
The objective is to analyze home value dynamics, identify long-term market trends, and build a reproducible data analysis workflow.

The analysis focuses on the Dallas–Fort Worth metropolitan area and surrounding cities.

## Project Goals

- Explore housing price trends in North Texas
- Build a structured data analysis workflow using Python
- Transform raw real estate datasets into an analysis-ready format
- Generate insights on long-term market movements

## Data Sources

The analysis uses publicly available housing market data from:

- Zillow Research Data

Raw datasets are not stored in the repository due to file size limitations.


## Tools and Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter / Google Colab
- Git & GitHub

## Key Topics Explored

- Housing price evolution over time
- Regional housing market trends
- Data transformation from wide to long format
- Time-series exploration of home values
